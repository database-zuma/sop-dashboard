FOLDER SOP
==========
Upload file PDF SOP ke folder ini.

Cara upload:
1. Buka https://github.com/database-zuma/sop-dashboard
2. Klik folder "sop"
3. Klik "Add file" > "Upload files"
4. Drag & drop file PDF
5. Klik "Commit changes"

Setelah upload, link PDF akan menjadi:
https://database-zuma.github.io/sop-dashboard/sop/nama-file.pdf

Gunakan link tersebut di pdfUrl pada sopData di index.html
